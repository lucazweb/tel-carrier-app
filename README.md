# TelecomApp

### Instalação ⚡️

Execute os seguintes comandos:

```
$ git clone https://github.com/lucazweb/tel-carrier-app
```

Acesse a pasta do projeto, e na raiz, execute o comando `npm run full-install`

(Se preferir pode instalar individualmente a mock-api e app front executando os comandos `npm install` na raiz do projeto e na pasta /client para instalar:)

### Executando o projeto

Na raiz do projeto, executar `npm run dev`
